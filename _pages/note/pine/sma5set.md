## SMA3×5+BB

- 目的
  - 一つのインジケータで3本の移動平均線とボリンジャーバンドを描画
- 特徴
  - 最大3本（短期・中期・長期）の移動平均線を出力します。
  - 期間は5セット登録できます
  - SMAは指定の時間足で登録したセットを描画します。  
- Source code
  - [【SMA3×5+BB - pjdhiro】](https://jp.tradingview.com/script/y6BTRb50-SMA3-5-BB-pjdhiro/)

## 2. マップ②：機動性・直進安定性・オフロード性能（3指標・スペック算出）

2026-07-12にユーザー指摘で全面改訂。**旧版は「取り回し力＝ホイールベース＋キャスター＋車重＋シート高」だったが、これは実は直進安定性の指標で、機動性（狭所Uターン）とは別物**。実測データで検証したら旧版は逆方向を向いていた（GB350を機動性最下位に誤判定＝下記）。3つの独立指標に分離した。

### 2-A. 機動性 ＝ 最小回転半径（実測優先）
取り回し・Uターンのしやすさは、ホイールベース単体ではなく**最小回転半径**で見るのが正しい（切れ角・WB・トレール・タイヤ径を織り込んだ完成値）。Honda・Kawasaki・Suzukiは公表、**ヤマハと外車は非公表**。

確認できた実測値（Honda公表）:
| モデル | ハンドル切れ角 | 最小回転半径 | 出典 |
|---|---|---|---|
| GB350 | 43°（公表）| **2.3m** | [young-machine](https://young-machine.com/2021/03/31/181642/) |
| CL250 | 38°（公表）| **2.6m** | [価格.comマガジン](https://kakakumag.com/car/?id=20296) |
| レブル250 | 35°（公表）| **2.8m** | [bike-news 最小回転半径解説](https://bike-news.jp/post/333367) |
| CRF250L | 非公表 | **2.3m** | [Honda公式](https://www.honda.co.jp/CRF250L/) |

- **これがGB350の誤判定を正した決定的証拠**：GB350はWB1440mmと長めでも切れ角43°で回転半径2.3m＝最小級。旧モデルはWBだけ見て最下位に沈めていた（正解は最上位級）。CL250も切れ角38°でWBの長さを相殺している（Uターン楽との評判・[webオートバイ](https://www.autoby.jp/_ct/17651768)）。
- **Kawasaki（メグロS1・エリミネーター400）は公表があるはずだが、公式HTML・検索・fetchでは値を取得できず**（PDF諸元表のみと思われる）。今は推定で埋めている。値が取れたら差し替える。
- 実測が無い13台は物理式で推定：`回転半径 ≒ ホイールベース ÷ sin(ハンドル切れ角) ＋ 0.18m`。補正0.18mはHonda4台の実測との差から較正（式は実測を平均+0.18で再現）。切れ角は公表3台（GB350/CL250/レブル）以外は同ジャンル典型値（クルーザー32-35°／クラシック・ネイキッド40°前後／エンデューロ42°）を仮定。

### 2-B. 直進安定性
ホイールベース（長いほど安定）＋キャスター角（寝ているほど安定）＋重心の低さ（シート高が低いほど安定）を正規化平均。クルーザー（エリミネーター400・レブル250）が上位で、「アメリカンは低重心＋長WB＋寝たフォークで直進安定に振る（直線・高速向き）」という見立てと一致。※シート高は重心の粗い代理で、真の重心はエンジン搭載位置等にも依るため厳密ではない。

### 2-C. オフロード性能
タイヤ種別（ブロックタイヤ＝スクランブラー/エンデューロか、50%）＋前輪サイズ（17〜21インチ、30%）＋サスストローク（確認車のみ、20%）の加重平均。前バージョンから変更なし。

### 再現コード（geometry3.py 相当）

```python
names=list(data)
def norm(v):    lo,hi=min(v),max(v); return {n:10*(x-lo)/(hi-lo) for n,x in zip(names,v)}
def norm_inv(v):lo,hi=min(v),max(v); return {n:10*(hi-x)/(hi-lo) for n,x in zip(names,v)}
# 機動性: turning radius (confirmed R else WB/sin(steer)+0.18), smaller=better
R={n:(d["R"] if d["R"] else round(d["wb"]/1000/math.sin(math.radians(d["steer"]))+0.18,2)) for n,d in data.items()}
maneuver=norm_inv([R[n] for n in names])
# 直進安定性: WB + caster + low-seat
stability={n:v/3 for n,v in {n:norm([data[k]["wb"] for k in names])[n]+norm([data[k]["ca"] for k in names])[n]+norm_inv([data[k]["sh"] for k in names])[n] for n in names}.items()}
# オフロード: tire50% + front30% + susp20%

susp={n:(10.0 if data[n]["susp"] and data[n]["susp"]>=200 else 6.0 if data[n]["susp"] and data[n]["susp"]>=140 else 0.0) for n in names}
offroad={n:fw_s[n]*0.30+tire[n]*0.50+susp[n]*0.20 for n in names}
```

### 既知の限界
- **機動性の切れ角は公表3台以外は推定**。Kawasaki 2台の公表回転半径が取れれば実測に差し替え。外車勢は非公表のため切れ角推定→回転半径推定の二段推定。
- **直進安定性のシート高は重心の粗い代理**。真の重心・マスの集中は反映していない。
- **エルゴノミクス（ハンドル幅・視界・シート形状）は数値化していない**。CL250が「街乗りしやすい」評判なのに機動性5.4（回転半径2.6mで中位）に留まるのは、回転半径以外の扱いやすさ（アップライト姿勢・軽いクラッチ等）がこの指標外だから。数値と評判は併読する。

## 4. ジャンル分類の根拠

6分類：欧州クラシック／米国クラシック／スクランブラー／エンデューロ／ネイキッド／クルーザー。判断基準は前輪サイズ（19インチ=スクランブラー、21インチ=エンデューロ）とブランドの出自。

- **キャバレロ250→エンデューロ**の根拠：Fantic Motorは1968年創業のエンデューロ・トライアル専業メーカー（80年代にトライアル世界タイトル3回、現在もモトクロス/エンデューロ/ラリー/GPを走る唯一のイタリアンブランド）。前輪は19インチ（CL250と同格）だが、乾燥130〜140kgの軽さと専業ブランドのジオメトリでスクランブラーとエンデューロの中間に位置づけた。出典：[Fantic Motor沿革](https://cybermotorcycle.com/marques/fantic/)
- **スヴァルトピレン401→スクランブラー（ただしオフロード性能は低め）**の根拠：Husqvarna公式が「compact scrambler」と明記（スポークホイール・ブロックタイヤ・スキッドプレート）。ただし前輪は17インチとスポーツバイク並みに小さく、マップ②のオフロード性能スコアは低い。出典：[Cycle World Buyer's Guide](https://www.cycleworld.com/husqvarna/svartpilen-401/)
- **CRF250L**：国内で新車が買える唯一の250ccオフロード車（他社は125cc以下かモタード系のみ）。出典：[webオートバイ 2026年モデル図鑑](https://www.autoby.jp/_ct/17831766)
- **セロー250**：2019年12月生産終了、2020年発売のファイナルエディションが最終モデル。以降は中古のみ。出典：[各種中古相場サイト](https://www.goobike.com/maker-yamaha/car-serow250/index.html)

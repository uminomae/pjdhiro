## 2. マップ②：取り回し力・オフロード性能の算出式

主観推定ではなく上表のスペックから機械的に算出。再計算・車種追加時は以下のPythonで再現できる。

**改訂履歴（2026-07-12）**：初版はオフロード性能を前輪サイズ主体にしていたが、それだと街乗り専用のメグロS1（18インチ・ストリートタイヤ）とスクランブラーのスヴァルトピレン401（17インチだがブロックタイヤ）が同スコアになる不整合が発生（ユーザー指摘）。タイヤ種別を主要因（50%）に組み替えて解消。同時に取り回し力へシート高（重心の代理指標）を追加——レブル250のシート高690mmが正しく評価されるようになった。

```python
import math

# name: (wheelbase_mm, caster_deg, weight_kg, seat_height_mm, front_wheel_in, susp_front_mm_or_None)
data = {
    "ブリット350":        (1390, 24.0, 195, 805, 19, None),
    "クラシック350":       (1390, 24.0, 195, 805, 19, None),
    "メテオ350":          (1400, 24.0, 191, 765, 19, 130),
    "GB350":             (1440, 27.5, 179, 800, 19, None),
    "スクランブラーSixty2":  (1460, 24.0, 183, 790, 18, None),
    "メグロS1":            (1415, 27.0, 143, 740, 18, None),
    "スピード400":         (1377, 24.6, 170, 790, 17, None),
    "スクランブラー400X":    (1420, 23.2, 180, 835, 19, 150),
    "スヴァルトピレン401":    (1368, 24.0, 159, 820, 17, 150),
    "CL250":             (1485, 27.0, 172, 790, 19, 150),
    "レブル250":           (1490, 28.0, 171, 690, 18, None),
    "レオンチーノ250":       (1380, 25.0, 162, 800, 17, None),
    "キャバレロ250":        (1410, 26.0, 138, 820, 19, 150),
    "ハーレーX350":         (1410, 24.8, 195, 817, 17, None),
    "エリミネーター400":     (1520, 30.0, 176, 735, 18, None),
    "CRF250L":           (1440, 27.0, 141, 830, 21, 230),
    "セロー250":           (1360, 26.5, 133, 830, 21, 230),
}
genre = {  # オフロード性能のタイヤ種別ボーナス判定に使用
    "ブリット350":"クラシック","クラシック350":"クラシック","メテオ350":"クラシック","GB350":"クラシック",
    "スクランブラーSixty2":"スクランブラー","メグロS1":"クラシック","スピード400":"ネイキッド",
    "スクランブラー400X":"スクランブラー","スヴァルトピレン401":"スクランブラー","CL250":"スクランブラー",
    "レブル250":"クルーザー","レオンチーノ250":"スクランブラー","キャバレロ250":"エンデューロ",
    "ハーレーX350":"ネイキッド","エリミネーター400":"クルーザー","CRF250L":"エンデューロ","セロー250":"エンデューロ",
}

names = list(data.keys())
wb  = [data[n][0] for n in names]; ca = [data[n][1] for n in names]
wt  = [data[n][2] for n in names]; sh = [data[n][3] for n in names]
fw  = [data[n][4] for n in names]; susp = [data[n][5] for n in names]

def norm_inv(vals):  # 小さい/軽いほど高スコア(0-10)
    lo, hi = min(vals), max(vals)
    return [10*(hi-v)/(hi-lo) for v in vals]

def norm(vals):  # 大きいほど高スコア(0-10)
    lo, hi = min(vals), max(vals)
    return [10*(v-lo)/(hi-lo) for v in vals]

# 取り回し力 = ホイールベース・キャスター角・車重・シート高（重心の代理指標）の正規化平均
wb_s, ca_s, wt_s, sh_s = norm_inv(wb), norm_inv(ca), norm_inv(wt), norm_inv(sh)
handling = [(wb_s[i]+ca_s[i]+wt_s[i]+sh_s[i])/4 for i in range(len(names))]

# オフロード性能 = タイヤ種別(50%、ブロックタイヤ=スクランブラー/エンデューロ genre) + 前輪サイズ(30%) + サス確認ボーナス(20%)
fw_s = norm(fw)
tire_s = [10.0 if genre[n] in ("スクランブラー","エンデューロ") else 0.0 for n in names]
susp_s = [(10.0 if s and s>=200 else 6.0 if s and s>=140 else 0.0) for s in susp]
offroad = [fw_s[i]*0.30 + tire_s[i]*0.50 + susp_s[i]*0.20 for i in range(len(names))]
```

### 既知の限界（次回編集時に思い出すこと）

## 4. ジャンル分類の根拠

6分類：欧州クラシック／米国クラシック／スクランブラー／エンデューロ／ネイキッド／クルーザー。判断基準は前輪サイズ（19インチ=スクランブラー、21インチ=エンデューロ）とブランドの出自。

- **キャバレロ250→エンデューロ**の根拠：Fantic Motorは1968年創業のエンデューロ・トライアル専業メーカー（80年代にトライアル世界タイトル3回、現在もモトクロス/エンデューロ/ラリー/GPを走る唯一のイタリアンブランド）。前輪は19インチ（CL250と同格）だが、乾燥130〜140kgの軽さと専業ブランドのジオメトリでスクランブラーとエンデューロの中間に位置づけた。出典：[Fantic Motor沿革](https://cybermotorcycle.com/marques/fantic/)
- **スヴァルトピレン401→スクランブラー（ただしオフロード性能は低め）**の根拠：Husqvarna公式が「compact scrambler」と明記（スポークホイール・ブロックタイヤ・スキッドプレート）。ただし前輪は17インチとスポーツバイク並みに小さく、マップ②のオフロード性能スコアは低い。出典：[Cycle World Buyer's Guide](https://www.cycleworld.com/husqvarna/svartpilen-401/)
- **CRF250L**：国内で新車が買える唯一の250ccオフロード車（他社は125cc以下かモタード系のみ）。出典：[webオートバイ 2026年モデル図鑑](https://www.autoby.jp/_ct/17831766)
- **セロー250**：2019年12月生産終了、2020年発売のファイナルエディションが最終モデル。以降は中古のみ。出典：[各種中古相場サイト](https://www.goobike.com/maker-yamaha/car-serow250/index.html)

## 6. 未解決・今後の課題

- ハーレーX350のスペック出典が個人ブログのみ。一次資料（Harley-Davidson公式）で裏取りできていない。
- キャスター角が推定の7台（§1参照）は、公式カタログPDFが見つかれば実測値に差し替える。
- 「積載」評価（釣り章）は主観のまま——リアキャリア対応可否など、メーカー公式アクセサリーカタログを当たれば一部は客観化できる。
- オフロード性能のサスストロークは確認済みが5台（メテオ350・400X・スヴァルトピレン401・CL250・キャバレロ250・CRF250L・セロー250、うち一部は姉妹車推定）のみ。残り10台は`None`扱い（サスボーナス0点）——実測値が見つかれば精度が上がる。
- 取り回し力・オフロード性能の重み付け（4等分平均、50/30/20%）は根拠となる統計的検証はしておらず、実測値を正規化した上での妥当な一次近似という位置づけ。異論があれば§2のPythonの重みを調整して再計算する。

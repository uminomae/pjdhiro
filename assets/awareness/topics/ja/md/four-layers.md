## 2. モデルの骨格

Layer 0 は、
呼吸、緊張、心拍のような身体状態の足場に関わる層です。
ここでは、
意識が最初から抽象的な思考として始まるのではなく、
身体の状態を土台に立ち上がることが重視されます。
重要なのは、
Layer 0 自体がすでに問題判断をしているわけではなく、
一人称経験の身体的足場を与えている点です。

Layer 1 は、
予測と現実の差を照合する層です。
すべての差がそのまま苦しさになるわけではなく、
ここでは差の検出と選別が起きていると考えます。
多くの差は前意識的に吸収され、
一部だけが違和感や欠けの芽として残ります。

Layer 2 は、
その差が何に関わる問題として受け止められるかを方向づける層です。
生存に関わる問題なのか、
信頼や関係に関わる問題なのかによって、
情動の向きと行動準備の向きが変わります。

Layer 3 は、
方向づけられた反応をすぐ固定せず、
見直す余地を保つ層です。
ここがあることで、
反応は自動的な短絡だけで終わらず、
再評価へ開かれます。

# What Is Awareness? - Survival and Intersubjectivity

## Purpose

This guide provides a high-level explanation of how the project currently frames awareness.
It is not a finished theory.
Its purpose is to present the current hypothesis, the research assumptions behind it, and the main questions that remain open.

## Executive Summary

- The current working hypothesis is that human experience can be read along two directions: survival and intersubjectivity.
- Awareness is tentatively described as a process that moves from bodily state, to discrepancy, to interpretation, to room for revising response.
- Neurophenomenology and psychology are the main reference fields, but their integration is still under review.

## Model Overview

The model starts from the hypothesis that two orientations matter when we read experience.

- Survival: avoiding danger, securing safety, protecting oneself
- Intersubjectivity: relationship, trust, belonging, being understood

To make that hypothesis readable, the project also uses a four-layer model.

- Layer 0: bodily state
- Layer 1: discrepancy between prediction and reality
- Layer 2: what kind of problem that discrepancy becomes
- Layer 3: room to avoid immediate fixation of response

The same event can therefore be interpreted differently depending on whether it is read mainly as a safety issue or as a relationship issue.

## Research Basis and Reference Fields

This framework is not presented as intuition alone.
The main reference fields are neurophenomenology and psychology.

From neurophenomenology, the project asks how first-person experience can be described in workable units.
From psychology, it asks how attachment, development, trust, and intersubjectivity can help test the relational side of the model.

Earlier source bundles are treated as historical material rather than as current canonical texts.
The current public explanation is organized around the survival-trust axis and the four-layer model.

## Current View

What currently looks most plausible is to read awareness not as a static container but as a sequence.
Bodily state comes first, discrepancy becomes salient, that discrepancy is interpreted in a particular direction, and some degree of response flexibility may or may not remain.

Consider a conversation in which someone suddenly becomes tense.
Instead of stopping at "the person became anxious," this framework asks a more structured set of questions.
Is the body already under strain?
Did an unexpected event occur?
Is the person reading the moment as a threat to relationship?
Has the response already become fixed before reflection is possible?

That kind of distinction can make both self-observation and observation of others more precise.

## Key Limitations and Next Questions

The research is still in progress.
Public guides, survey documents, and component reports are in place, but the link between neurophenomenology and psychology is not yet fully organized.

The main open questions are:

- How clearly can survival-oriented and relationship-oriented problems be separated?
- Is the four-layer model sufficient as a minimal description of awareness?
- How far can response flexibility be described as a stable structural feature?
- At what point should neurophenomenology and psychology be connected?
- How far can these hypotheses be translated into observable terms?

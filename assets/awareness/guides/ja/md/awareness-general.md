# 意識とは何か — 生存と間主観性を手がかりに

## 概要

この文書は、`awareness-space` が扱っている意識モデルを、
初めて読む人にも全体像がつかめるように説明するための guide です。
ここで中心に置くのは、
意識が身体状態、予測と現実の差、情動の方向づけを通じて
どのように立ち上がるかという見取り図です。
特にこの repo では、生存と間主観性をコア仮説として扱い、
それが本当に有効な見方なのかを確かめようとしています。

## 意識を流れとして見る

このモデルでは、意識を4つの層の流れとして見ます。

### Layer 0

体の状態の層です。
呼吸、緊張、心拍、落ち着かなさのような足場がここにあります。
人は何かを考える前から、すでに体の状態の影響を受けています。

### Layer 1

予測と現実の差が前景化する層です。
思っていた通りに進んでいるか、予想外のことが起きたかがここで目立ちます。

### Layer 2

その差が、何に関わる問題として受け止められるかを読む層です。
ここで役に立つのが F-O軸です。

- F軸は、生存、安全、防衛の方向です
- O軸は、関係、所属、信頼、間主観性の方向です

同じ出来事でも、F軸として読むか O軸として読むかで、
立ち上がる情動や次の反応は変わります。

### Layer 3

方向づけられた反応が、そのまま外へ出るとは限りません。
人は反応の前に揺れ、見直し、別の読み方へ移る余地を持つことがあります。
この repo では、その余地がどのような条件で成立するのかを、
補助的な論点として扱っています。

## この見方で何がわかるか

このモデルの利点は、
気分や反応を「性格だから」で終わらせず、
どこで何が起きているかを分けて見やすくなることです。

たとえば焦りがあるとき、
それは体の緊張が強いのか、
予想外のことが起きたのか、
関係の不安として受け止めているのか、
それとも見直す余地を持てないまま反応へ流れているのか、
という読み分けができます。

## 何の役に立つか

- 自分の反応を少し離れて見やすくなる
- 他者の反応を単純に良し悪しで見なくなる
- いま何が起きているかを観察する足場になる

## 確かなことと未解決のこと

現時点で確かなのは、
体の状態、予測との差、情動の方向づけという観点を分けると、
意識経験を整理しやすいということです。

一方で、F軸と O軸の境界をどこまで明確に引けるか、
それが意識モデルの中心仮説として妥当かどうかは、
まだこれから詰めていく必要があります。

# 意識モデルと神経現象学・心理学の接続

## 序論

本稿は、`awareness-space` が扱う意識モデルを、
神経現象学と心理学を中心軸にして整理する探索的 guide である。
目的は、完成理論を宣言することではない。
どの仮説を中心に置き、
その仮説を確かめるためにどの調査を参照し、
現時点で何が見えていて、
何がまだ未整理なのかを明示することである。
また、本稿自体は summary であり、
詳細な reader-facing な論拠は REPORTS、
進行中の調査設計は survey と design memo に分かれている。

## モデルの中心仮説

現時点での起点仮説は 生存-信頼軸である。

- 生存軸: 生存、安全、防衛、保身
- 信頼軸: 関係、信頼、所属、間主観性

この二軸は感情分類の一覧ではなく、
差がどこへ向けて意味づけられるかを読むための仮説座標系として置かれている。

同時に、意識は4層の処理連鎖として記述するのが有力だと考えている。

1. Layer 0: 体の状態
2. Layer 1: 予測と現実の差
3. Layer 2: 差の方向づけ
4. Layer 3: 応答が固定されず見直されうる余地

4層モデルの原著文書群はこの流れを文章として支える基盤 source であり、
Concept Notes は補助論点と境界事例の束として位置づけられる。
一方、反応を保持し見直す機能は重要な周辺仮説ではあるが、
現時点では中心確定概念としては扱っていない。

## 調査設計

このモデルが中心参照にしているのは神経現象学と心理学である。

神経現象学に求めているのは、
一人称経験をどのような単位で記述できるかという問いである。
身体状態、気づき、前景化、応答以前の揺れをどう捉えるかは、
この軸で読むべき中核問題になる。

心理学に求めているのは、
生存-信頼軸、特に 信頼軸側の妥当性をどこまで検証できるかという問いである。
そのために、発達心理学、愛着、信頼、間主観性、社会的認知の知見を重視している。

したがって、このモデルは
「神経現象学が経験の記述単位を支え、心理学が関係形成と方向づけの検証軸を支える」
という分業を仮定しながら進めている。

ただし、この仮定には段階差がある。
生存-信頼軸、4層モデル、Concept Notes には
reader-facing な report がある。
一方で、神経現象学と発達心理学については、
中心参照として重要であるにもかかわらず、
現時点では専用 report より design memo の段階が先行している。

また、`抱持` は独立概念としての境界がまだ粗く、
現在は概念監査中の仮ラベルとして扱う。

## 現時点で何が見えているか

現段階で比較的強く見えているのは、
意識を静的内容ではなく、
身体状態から差の前景化、方向づけ、応答の余地へ至る流れとして読む見方である。

この枠組みを取ることで、
情動、認知、行動準備を別々の箱に分けず、
一つの連鎖として記述できる可能性が出てくる。
また、生存-信頼軸を加えると、
同じ差が安全の問題として読まれているのか、
関係の問題として読まれているのかを区別しやすくなる。

さらに、歴史的 source 束を読むと、
既存素材は断片的ではあるが、
4層モデル、生存-信頼軸、関係性、信頼、測定といった論点をすでにかなり保持していることも見えている。

したがって、ここでの「見えていること」は、
主として公開済み report と source map に支えられた整理である。
他方、神経現象学と発達心理学との厳密な統合条件まで
reader-facing に記述し終えたとはまだ言えない。

## どこまで進んでいるか

しかし、source が厚いことと、
統合理論が完成していることは同じではない。

現時点では、
生存-信頼軸、4層モデル、Concept Notes を起点にした
有力な構造仮説は成立している。
その一方で、主質問の粒度はまだ粗く、
神経現象学と心理学をどの結節で統合するかは未設計の部分が多い。

公開している guide や構成要素レポートも、
完成版というより公開ドラフトの段階にある。
したがって、現在の状態は
「理論がない」のではなく、
「有力な構造仮説と厚い source はあるが、統合の仕方がまだ固まり切っていない」
と表現するのが最も正確である。

より具体的には、

- report-backed な公開整理は、生存-信頼軸、4層モデル、Concept Notes にある
- survey-backed な現在地整理は `survey-status.md` にある
- 神経現象学 / 発達心理学は、中心的だが design memo 段階の比重がまだ大きい
- `抱持` は report-backed な独立概念ではなく、再調査と概念監査の対象である

## 未解決の問い

未解決の論点は少なくとも次の通りである。

- 生存軸と 信頼軸の境界をどこまで厳密に定義できるか
- 4層モデルが意識記述の最小単位として十分か
- Layer 1 の差を、どこまで中立的に記述できるか
- 応答を見直す余地を、独立機能として扱うべきか、循環条件として扱うべきか
- 神経現象学と発達心理学をどこまで一つの構造に統合できるか
- 観測可能性をどこに置くか

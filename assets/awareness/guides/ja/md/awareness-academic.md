# 意識モデルと神経現象学・発達心理の接続

## 序論

本稿は、`awareness-space` が扱う意識モデルを、
神経現象学と発達心理を中心とする知見との接続の中で記述するための
探索的 guide である。
ここで目指すのは完成理論の宣言ではなく、
どの部分が比較可能で、どの部分が未整理なのかを明示することである。

## モデルの骨格

このモデルの中心には 4層モデルがある。

1. Layer 0: 内受容感覚
2. Layer 1: 予測と現実の差
3. Layer 2: F-O軸による方向づけ
4. Layer 3: 応答の可変性

この順序は、意識を静的な内容の集まりではなく、
身体状態から方向づけまでの連鎖として捉えるための足場になる。

## 比較の観点

### 1. 身体状態の位置づけ

意識経験を説明するモデルは、
しばしば高次認知や意味づけを前景化する。
しかし本モデルでは、体の状態を最初の層として置く。
この点は、情動や注意の議論を身体状態から考える立場との接続点を持つ。

### 2. 予測との差

Layer 1 は、
世界がどうであるかを受け取る層というより、
想定との差が前景化する層として置かれる。
ここでの要点は、差そのものの検出であり、
その差がどんな意味を持つかはまだ確定していないということである。

### 3. F-O軸

Layer 2 では、
その差が何に関わる問題として受け止められるかが方向づけられる。
F軸は安全、防衛、保身の方向であり、
O軸は関係、信頼、所属、間主観性の方向である。

この二軸は感情カテゴリの一覧ではない。
むしろ、情動価と行動準備がどこへ向くかを整理するための座標系として扱うべきである。

### 4. 応答の可変性

Layer 3 では、方向づけられた反応がそのまま外へ出るのではなく、
場や関係の条件によって変わりうる点に注目する。
この repo では、その具体的な理論化はまだ周辺論点であり、
中心仮説としては扱わない。

## 構造として読んだときの含意

このモデルを流れとして読むと、
意識は内容の所有ではなく、
差の検出、方向づけ、反応保留の連鎖として見えてくる。
その結果、情動、認知、行動準備を分断せずに記述できる可能性がある。

同時に、未整理の点も残る。
F-O軸が二軸で十分なのか、
生存と間主観性をコアとする見方が十分な説明力を持つのか、
観測可能性をどこに置くかは、なお開いた問いである。

## 未解決の問い

- 教育、支援、臨床、組織へ接続するときに何が保たれ、何が変形するか

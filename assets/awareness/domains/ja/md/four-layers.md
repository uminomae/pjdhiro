# 4層モデルに関する報告

## 1. この文書の目的

この文書は、
意識を静的な内容の集まりとしてではなく、
どのような作動の流れとして記述できるかを説明するためのものです。
本稿ではその最小構造を、
4つの層から成る連鎖として整理します。

## 2. モデルの骨格

Layer 0 は、
呼吸、緊張、心拍のような身体状態の足場に関わる層です。
ここでは、
意識が最初から抽象的な思考として始まるのではなく、
身体の状態を土台に立ち上がることが重視されます。

Layer 1 は、
予測と現実の差を照合する層です。
すべての差がそのまま苦しさになるわけではなく、
ここでは差の検出と選別が起きていると考えます。

Layer 2 は、
その差が何に関わる問題として受け止められるかを方向づける層です。
生存に関わる問題なのか、
信頼や関係に関わる問題なのかによって、
情動の向きと行動準備の向きが変わります。

Layer 3 は、
方向づけられた反応をすぐ固定せず、
見直す余地を保つ層です。
ここがあることで、
反応は自動的な短絡だけで終わらず、
再評価へ開かれます。

## 3. なぜ重要か

このモデルの利点は、
同じ「つらさ」や「焦り」を一つの塊として扱わずに済むことです。
身体状態の揺れなのか、
差の検出なのか、
方向づけの問題なのか、
それとも見直す余地を失っているのかを、
段階ごとに読み分けられます。

また、
この見方は情動と認知を分断しません。
情動は方向づけの中で立ち上がり、
認知的な見直しは反応を固定しない余地によって支えられます。
意識を感覚か思考かの二択にせず、
複数の層が連動する動態として捉えられる点が重要です。

## 4. 現時点で言えること

現時点では、
身体状態、差の検出、方向づけ、見直しの余地という順序で意識を読む見方は、
かなり有力です。

特に重要なのは、
生存-信頼軸と反応の見直しの余地が、
身体と差の上に後続的に立ち上がるという点です。
これにより、
苦しさや緊張を単なる気分としてではなく、
処理のどこで何が起きているかとして捉えやすくなります。

## 5. どこまで確定しているか

4層モデルは有力な整理ですが、
まだ細部まで固まっているわけではありません。
とくに、
Layer 1 の差のうちどこからを問題の立ち上がりとみなすか、
Layer 3 を独立した層として扱うべきかは、
なお検討が必要です。

また、
各層を観察可能な記述や指標へどう接続するかも、
今後の重要な課題です。

## 6. 今後の検討課題

- Layer 1 の差をどこまで問題の立ち上がりとして扱うか
- Layer 2 に第三の軸が必要か
- Layer 3 を独立層とみるか、再評価条件とみるか
- 各層を観察可能な記述へどう接続するか
- Layer 0 と Layer 1 の境界をどこまで精密に書けるか

## 7. まとめ

4層モデルは、
意識を身体状態から反応の見直しまでの流れとして捉えるための、
現時点で最も有力な枠組みの一つです。
強みは、
ズレ、情動、行動準備、見直しの余地を
一つの連鎖として見渡せる点にあります。

今後は、
各層の境界と観察可能性をさらに詰めることで、
この枠組みをより安定した記述へ近づける必要があります。

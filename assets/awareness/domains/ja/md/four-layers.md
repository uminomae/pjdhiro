# 4層モデルに関する調査報告

この report では、4層モデルの reader-facing な骨格説明と、同ディレクトリに置いた原著文書群をまとめて扱う。

## 1. この report の問い

本 report の問いは、意識を「分かったあとに語る内容」ではなく、どのような作動の連なりとして記述できるかである。`awareness-space` では、その最小構造を Layer 0 から Layer 3 までの4層で捉える。

## 2. モデルの骨格

Layer 0 は内受容感覚の層であり、呼吸、緊張、心拍のような身体状態の足場を与える。ここで重要なのは、意識が最初から抽象的な思考ではなく、身体状態の布置の上に立ち上がるという点である。Layer 0 自体はまだ欠損を作らないが、身体状態表象を Layer 1 に渡すことで、欠損が生じる前提条件を与える。

Layer 1 は予測と現実のズレを照合する層である。すべてのズレが欠損になるわけではなく、ここではズレの検出と選別が起きていると考える。多くのズレは前意識的に吸収され、一部だけが欠損の芽として次層へ送られる。

Layer 2 は生存-信頼評価の層である。Layer 1 から送られたズレは、生存と信頼のどちらに関わるものとして受け止められるかによって、情動価と行動準備の向きが変わる。

Layer 3 は Withhold の層である。ここでは、すでに方向づけられた行動準備をそのまま反応へ流さず、一時的に保持することで再評価の余地を作る。

## 3. このモデルで現在言えること

この4層モデルの強みは、意識を静的な箱ではなく処理連鎖として読めることにある。身体状態の足場、ズレの検出、価値づけ、保持という順序を切り分けることで、同じ「つらさ」でもどの層で何が起きているかを見分けやすくなる。

また、このモデルは情動と認知を分断しない。情動は Layer 2 の評価として立ち上がり、認知的な見直しは Layer 3 の保持と再評価に支えられる。意識は感覚か思考かの二択ではなく、複数の層が連動する動態として扱われる。

さらに、Layer 0 と Layer 1 の読解を踏まえると、このモデルは少なくとも次の点を支えている。

1. 意識経験は身体性から切り離せない
2. 欠損は単なるズレの計算ではなく、一部のズレが立ち上がった一人称経験である
3. 生存-信頼の評価と Withhold は、身体とズレの上に後続的に立ち上がる

## 4. 一人称経験と三人称記述の境界

`awareness-space` では、4層モデルを読むときに一人称記述と三人称記述を混同しない。

- 一人称記述: 何かが足りない、違和感がある、逃げたい、問いたい
- 三人称記述: 予測誤差信号、precision weighting、島皮質、vmPFC などの説明語彙

4層モデルの主語は前者である。後者はモデルを照らす参照枠として使うが、それ自体が本論ではない。

この区別により、Layer 1 は「意識化された予測誤差」そのものではなく、欠損が立ち上がる入口として読むことができる。Layer 0 も同様に、身体性の足場を与えるが、それだけで一人称経験を尽くすわけではない。

## 5. 主要な含意

第一に、欠損は Layer 1 のズレ検出だけでは完結しない。Layer 2 で生存-信頼の方向づけを受けて初めて、何が耐えがたく、何を守るべきかが立ち上がる。

第二に、Withhold は付け足しの徳目ではなく、4層モデル全体を探索へ開くゲートとして機能する。保持がなければ、評価は反応へ短絡しやすい。

第三に、このモデルは4層モデル原著文書群の各パートを再編する受け皿としても使える。各パートは、4層のそれぞれを厚く記述した素材として読み直せる。

第四に、このモデルは 生存-信頼軸と Withhold を別々の論点として置き去りにせず、同じ処理連鎖の中で位置づけられる。生存-信頼軸は Layer 2 の評価として、Withhold は Layer 3 の保持として、それぞれの役割が見えやすくなる。

## 6. まだ決めていないこと

- Layer 1 でどこまでを「欠損」と呼ぶか
- Layer 2 の生存-信頼評価に第三の軸が必要か
- Layer 3 を単独層として扱うべきか、循環全体の再評価機構として扱うべきか
- 各層を観察可能な指標へどう接続するか
- Layer 0 と Layer 1 の境界を、前反省的経験の記述としてどこまで精密化できるか

## 7. まとめ

4層モデルは、意識を身体状態から保持機能までの処理連鎖として捉えるための最小フレームである。強みは、ズレ、情動、行動準備、問いの保持を一つの流れとして見渡せる点にある。

現時点では特に、Layer 0 と Layer 1 の読解結果を受けて、意識が身体性とズレの選別の上に立ち上がるという輪郭が前より明確になった。今後は Layer 2 と Layer 3 をさらに厚くすることで、生存-信頼軸と Withhold をこのフレームの中でどこまで精密に記述できるかを詰めていく。

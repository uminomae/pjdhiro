# Phase 9 — 深層探索計画

![Phase 9 深層探索計画 — 計画概要インフォグラフィック](https://uminomae.github.io/pjdhiro/assets/creation/img/svg/phase9/ja/overview-01-overview-svg.svg)

## 位置づけ

Phase 8 の領域横断分析で抽出された5つの構造テーマを、さらに深く掘り下げる段階です。Phase 8 が「30領域に共通する構造パターンの発見」であったのに対し、Phase 9 は「その発見がどこまで確かで、どこまで通用し、どこまで形式化できるか」を検証します。

## 4トラック概要

| トラック | 名称 | 目的 |
|---------|------|------|
| 9A | 実証接地 | 構造テーマを学術知見と突合し、根拠ある主張と未検証の主張を分離する |
| 9B | 反証探索 | 確証バイアスを排除するため、積極的に反例と破綻ケースを探索する |
| 9C | 未踏領域探索 | 30領域外にモデルを適用し、予測力と汎用性を検証する |
| 9D | 数理形式化 | 構造テーマを数学的に表現し、定理導出と予測生成を可能にする |

## 依存関係

```
9A 実証接地 ───→ 9B 反証探索
     │                │
     │                ↓
     └──────→ 9D 数理形式化

9C 未踏領域探索（独立して着手可能）
```

- **9A → 9B**: 実証接地の結果を踏まえて反証を探索するのが最も効果的
- **9A, 9B → 9D**: 修正済みの主張を形式化対象とする
- **9C**: 他トラックと独立に着手可能。ただし 9A/9B の知見があれば予測精度が向上

## 推奨着手順序

1. **9A 実証接地** — 最初に着手。縁の類型学（28領域）から開始
2. **9C 未踏領域探索** — 9Aと並行して着手可能
3. **9B 反証探索** — 9Aの成果が出た後に着手
4. **9D 数理形式化** — 9A/9Bの修正済み主張が揃った後に着手

## 現在の状態

全4トラックが計画段階にあります。各トラックの詳細な調査計画は個別のカードからご確認いただけます。

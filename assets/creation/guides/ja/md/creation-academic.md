# 1. 問いの設定

創造性研究、認知科学、組織論、精神分析、複雑系科学——これらの領域は、創造的プロセスに関して異なるラベルを使い、異なるスケールで記述を行ってきた。Wallasの4段階、Guilfordの発散/収束、Bionのnegative capability、Prigogineの散逸構造、野中のSECIモデル。これらは異なるものを記述しているのか、それとも同じ構造の異なる面を指しているのか。

本稿は、この問いに対して5段階モデル（場→波→縁→渦→束）を分析の枠組みとして提示し、30の学問領域を横断する調査の結果を報告する。分析の目的は以下の3点である。

1. 先行理論と5段階モデルが同じ構造を指しているかを検討する
2. ラベルで表現されていないもの——先行理論が見落としている構造——を同定する
3. 5段階モデル側の見落としを検討する

これは指月的記述の実践である。5段階モデルと先行理論は、それぞれ月を指す指である。指の優劣を論じるのではなく、同じ月を指しているかを分析する。

# 2. 分析の方法

## 2.1 調査の設計

30の学問領域を対象とし、各領域から10の理論・概念を選定した。選定基準は、創造的プロセスに関する段階的記述または構造的記述を含むこと、領域内で一定の影響力を持つこと、の2点である。

## 2.2 評価手法

各理論について、5段階モデルとの構造対応をE-1〜E-8の8項目で評価した。評価は四面記述——類似、独自、学び、文脈——で構造化した。

| 面 | 記述内容 |
|----|---------|
| 類似 | 5段階モデルと同じ構造を指していると考えられる部分 |
| 独自 | 一方にはあるが他方にはない構造 |
| 学び | 先行理論から取り込むべき知見 |
| 文脈 | 各理論の固有文脈。本論の用語に翻訳しない |

品質チェックとして、非二元性チェック（安易な同一視の防止）、フラクタルチェック（スケールの混同防止）、群盲象チェック（3領域以上での確認）を適用した。

## 2.3 方法論的限界

本分析には以下の限界がある。第一に、評価者はAI（複数のモデルによる多角的評価）と著者であり、各領域の専門家による検証は行われていない。第二に、構造対応の判定は定性的であり、統計的な検証は行っていない。第三に、30領域の選定自体に著者のバイアスが含まれる可能性がある。

これらの限界は、本分析が仮説生成のフェーズにあることを意味する。仮説の検証は今後の課題であり、本稿の範囲外である。

# 3. 5段階モデルの定義

5段階モデルは、創造を「よいアイデアを出す能力」ではなく、ある状態から別の状態への連続的な変換として記述する。

| 段階 | 名称 | 構造 | プロセス | 生成されるもの |
|------|------|------|---------|--------------|
| 1 | 場（Field） | ゼロ・無 | 静寂 | 可能性の母体 |
| 2 | 波（Wave） | ゆれ・対立 | 分離 | 差・方向性 |
| 3 | 縁（Relation） | 境界・関係 | 繋がり | ルール・制約・接続 |
| 4 | 渦（Vortex） | 個・立ち上がり | 包摂・融合 | まとまり（プロトタイプ） |
| 5 | 束（Bundle） | 方向 | 集合 | 構造化された成果 |

5段階は直線ではなく循環する。束から場への回帰——生まれた構造を前提として次の違和感を漂わせること——が創造を継続させる。

本モデルの射程について付言する。5段階は「創造プロセス」のモデルとして提示しているが、調査の結果、自己組織化、個体化、相転移といった、より広い現象にも構造対応が確認された。物理的/生命的/心的個体化の区別はスペクトラム上のラベルにすぎない可能性がある。この射程の広さについては §6 で改めて検討する。

欠損駆動思考との接続点を述べる。場はゼロ・無の状態である。違和感すら生じていない。最初の微かな揺らぎが生じた時点で、波への移行が始まる。Withhold——反射的に処理せず、誤差を問いとして保持する機能——が作動するのは波以降である。波でF-O評価（生存軸・愛着軸）によって対立として立ち上がり、縁で複数の評価が交差し、渦で新しい構造として再統合され、束で再利用可能な知として安定する。

# 4. 先行研究との構造比較

## 4.1 Wallasの4段階モデル

Wallasの準備→孵化→照明→検証は、創造性研究における最も影響力のある段階モデルである。

| 面 | 分析 |
|----|------|
| 類似 | 孵化は場〜波に、照明は縁〜渦に、検証は束にそれぞれ対応する。意識的な作業と非意識的な処理の交代という基本構造が共通する |
| 独自 | 5段階モデルには「縁」という明示的な段階がある。Wallasの記述では、孵化と照明の間の移行——矛盾が最大化し、どちらにも収束しない状態——が構造的に記述されていない |
| 学び | Wallasの「準備」段階の重要性。5段階モデルの「場」は準備を含むが、意図的な知識獲得のプロセスが暗黙的になっている |
| 文脈 | Wallasのモデルは内省報告に基づいており、主に個人の思考プロセスを記述している。社会的プロセスは射程外である |

Wallasの記述に縁に対応する段階が明示的にあるかという問いに対しては、孵化の後期に構造的類似が見られるものの、独立した段階としては記述されていないと考えられる。

## 4.2 Guilfordの発散/収束

| 面 | 分析 |
|----|------|
| 類似 | 発散は場→波→縁の前半に、収束は渦→束に対応する。思考の方向性という基本的な区分が共通する |
| 独自 | 二分法では縁が消える。矛盾が最大化する段階——発散でも収束でもない状態——に名前がない |
| 学び | 発散的思考の測定手法（流暢性、柔軟性、独自性）は、波の段階の操作的定義として有用である |
| 文脈 | Guilfordのモデルは因子分析に基づく個人差研究の枠組みであり、プロセスの段階記述を主目的としていない |

この分析は、発散/収束の二分法が広く採用されてきた理由（操作的に定義しやすい、測定が容易）と、その代償（縁の不可視化）の両面を示している。

## 4.3 SECIモデル（野中・竹内）

| 面 | 分析 |
|----|------|
| 類似 | 共同化は場に、表出化は波〜縁に、連結化は渦に、内面化は束にそれぞれ対応する。暗黙知と形式知の変換という基本構造に類似がある |
| 独自 | SECIの「場（Ba）」は社会的コンテクスト（共有の場）であり、5段階の「場」（個人内の未分化状態）とは次元が異なる。名称の類似に惑わされてはならない |
| 学び | 組織的な知識創造プロセスとしての記述。5段階モデルは個人内のプロセスから出発しているが、SECIの社会的次元との接続が必要である |
| 文脈 | SECIは組織論・経営学の枠組みであり、個人の認知プロセスではなく組織的な知識変換を記述している |

## 4.4 フロー体験（Csikszentmihalyi）

| 面 | 分析 |
|----|------|
| 類似 | フロー体験における「挑戦とスキルのバランス」は、縁の段階における矛盾の保持と構造的に類似する |
| 独自 | フロー体験は段階ではなく、5段階のいずれの移行時にも生じうる進行の質として位置づけるのが適切である。フローは「どの段階にいるか」ではなく「段階間の移行がどのように進んでいるか」を記述する |
| 学び | 最適経験の条件（明確な目標、即時フィードバック、挑戦/スキルバランス）は、5段階の各移行がスムーズに進む条件の記述として参考になる |
| 文脈 | Csikszentmihalyiの研究は主観的経験の現象学的記述であり、構造モデルではない |

## 4.5 Bionのnegative capability

| 面 | 分析 |
|----|------|
| 類似 | negative capability——答えのない状態に留まる能力——は、縁の段階の核心と構造的に重なる。不確実性の中に留まることが創造の条件であるという洞察は、縁の作法と同型である |
| 独自 | 5段階モデルは、negative capabilityを単独の能力としてではなく、5段階の中の特定の段階（縁）における構造的な要請として位置づける。つまり「なぜ留まる必要があるか」に構造的な回答を与える |
| 学び | Bionのcontainer/contained概念は、縁に留まるための条件——矛盾を受け止める関係の質——を記述する補助概念として有用である |
| 文脈 | Bionは精神分析の臨床文脈で記述しており、個人の認知プロセスへの適用には解釈を要する |

## 4.6 複雑系科学（散逸構造、自己組織化）

| 面 | 分析 |
|----|------|
| 類似 | 散逸構造の形成過程——平衡から遠く離れた系が揺らぎを経て新しい秩序を生む——と5段階の間に強い構造対応が確認された。場は平衡状態、波は揺らぎの増幅、縁は臨界点付近の準安定状態、渦は新秩序の自発的形成、束は安定構造に対応する |
| 独自 | 物理的な自己組織化の記述には「縁に留まる」という意志的な要素がない。5段階モデルは、この段階にWithhold（保持）という意志的な機能を導入している |
| 学び | 臨界点付近の振る舞いに関する定量的な知見は、縁の段階の特性を理解するための補助線として有用である |
| 文脈 | 物理化学的な現象の記述であり、人間の創造プロセスとの対応は構造的類似の段階にある。この対応が構造的同一性なのか表面的類似なのかの検証は、未解決の問いである |

# 5. 横断的知見

30領域の調査を通じて、以下の横断的パターンが確認された。

## 5.1 縁に対応する段階の遍在

最も顕著な知見は、「矛盾が最大化し、どちらにも収束しない状態」が多くの領域で独立に記述されていることである。Bionのnegative capability、複雑系の臨界点、人類学のリミナリティ（Turner）、建築のin-between（Ando）——これらは異なるラベルで同じ構造を指している可能性が高い。

一方で、この段階を既存の段階モデルが明示的に記述していることは少ない。Wallasの4段階、Guilfordの発散/収束、デザイン思考のダブルダイヤモンド——いずれも縁に対応する段階を独立に命名していない。これは、縁が測定や操作化が困難な段階であることに起因すると考えられる。

## 5.2 「境界は厚い」

複数の領域で、創造的プロセスにおける境界領域が単なる移行点ではなく、固有の構造を持つ「厚い」領域であるという知見が確認された。複雑系科学における相転移の臨界領域、建築における壁の厚み（境界が場所になる）、人類学における通過儀礼のリミナル・フェーズ——これらは境界そのものが創造の場であることを示唆する。

## 5.3 循環構造の普遍性

束から場への回帰——安定した構造が次の不安定の起点になる——は、複雑系のフィードバックループ、SECIの知識スパイラル、生物学の発生と再編といった異なるスケールで観察される。直線的な段階モデル（Wallas）では、この循環が構造的に記述されにくい。

# 6. 根拠の状態と未解決の問い

## 6.1 確定していること

5段階の構造定義と、各段階の基本的な特性記述は確定的である。30領域中の多数の領域で構造対応が確認されたことは事実として報告できる。

## 6.2 有力だが検証途上のもの

縁が構造類似の判別力が最も高いという知見は、複数領域で一貫して確認されており、有力な仮説と考えられる。「境界は厚い」というパターンも同様である。

## 6.3 未解決の問い

以下は保持論点として明示する。

**スケールの問題**: 物理的な臨界現象と人間の創造的プロセスの間には、スケールの違いがあるように見える。この対応が構造的同一性なのか表面的類似なのかの検証は、本分析では完了していない。5段階モデルの射程——「存在そのものの創造」を記述しているという仮説——の検証は今後の最重要課題である。

**Withhold の操作的定義**: 「矛盾を保持し、反射的に処理しない」という記述は、操作的定義としてはまだ不十分である。どのような条件で保持が可能になり、どのような条件で破綻するのかの定量的な記述が必要である。

**社会的次元との接続**: 5段階モデルは個人内のプロセスから出発しているが、組織論（SECI）や社会心理学が記述する社会的次元との統合は未完了である。個人の「場→束」とチームの「場→束」の入れ子構造については、理論的な記述にとどまっている。

**5段階モデル側の見落とし**: 本分析は5段階モデルを枠組みとして使用しているため、5段階モデル自体が見落としている構造を検出しにくいという方法論的限界がある。この限界に対しては、先行理論の「独自」面（5段階にはないが先行理論にあるもの）の分析で部分的に対処しているが、十分ではない。

## 6.4 温度の開示

本分析の結果は、異なる領域の先行理論と5段階モデルの間に構造的な対応が広く確認されたことを報告するものである。この対応が「同じ構造を指している」と結論づけるには、個別領域の専門家による検証、定量的な分析手法の導入、および5段階モデル自体の操作的定義の精緻化が必要である。

確定していないことを隠すのではなく、根拠の状態をそのまま開示することが、本分析の誠実な報告のあり方である。

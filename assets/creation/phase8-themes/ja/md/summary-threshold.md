# T2: 閾値構造と縁から渦への遷移条件 — 結論要約

11ドメインが閾値遷移を直接主題化。加えて渦不到達経路を記述する10ドメインが関連する。

- **縁から渦への遷移は連続的でなく閾値的である**: 物理化学的に最も堅い横断知見の一つ
- **閾値を超えた遷移は不可逆的である**: 一度渦に入ると縁に戻れない。ただし系全体は循環する
- **閾値は「制御可能」な場合がある**: 触媒は閾値低下装置、Withholdは閾値を上げる態度
- **「渦に至らない経路」が存在する**: 5段階は「必ず渦に到達する」のではなく、縁で止まる経路も含む

「帯域としての縁」（T1の知見）と「閾値としての遷移」は緊張関係にある。帯域は幅を持ち閾値は点を指す。触媒（閾値を下げる）とWithhold（閾値を上げる）の対称性は、「縁に留まる」と「渦に入る」の制御メカニズムとして理論化できる可能性がある。

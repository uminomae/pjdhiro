### 円の作り方：単純作業を繰り返して作る

- 元気玉みたいにいろんなとこから集める。
  - どこから？：ジュリア集合(タツノオトシゴのような形など)のとこから
  - どんなふうに？：２乗と移動を繰り返して

```
# 反復（n = 0, 1, 2, …）
for n in 0, 1, 2, …:
    z[n+1] = (z[n])^2 + c
```

### 塗りつぶしてる"点"たちはどこからやってきたのか？:作った過程を逆戻り

作り方の逆。cの分だけ戻す。ルートする。繰り返す。

---

### パイの比喩

パイの上に粉砂糖をばらまき、回転（＝二乗）と傾き（＝平行移動）を繰り返します。

- 二乗操作（z → z^2）：  
  パイを高速で回転させると、粉砂糖の粒一つ一つは、パイの中心から見た向き（角度）が２倍になり、中心からの距離は二乗されるように飛び跳ねます。
- 定数 c（平行移動）：  
  パイは常に一定方向にわずかに傾き続けるため、回転後の粉砂糖は毎回その傾き分だけまとめて平行移動します。
- 発散判定：  
  回転と傾きの結果、パイの縁を越えて落ちた粉砂糖はこぼれて二度とパイの上には戻りません。

どの粒が最後まで残るかを見ます。残った粒のもとあった場所が Julia 集合に対応します。
